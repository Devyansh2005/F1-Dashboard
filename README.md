# Devyansh's Pit Wall · Your Personal F1 2026 Dashboard 🏎️🏁

🚀 **[View Live Demo](https://f1-dashboard-sepia.vercel.app/)**
Welcome to the **Pit Wall** — a fast, premium, and broadcaster-quality personal Formula 1 dashboard. 

This project distills the most critical information of a bustling race weekend into a single, beautifully designed dashboard. With no complex backend or databases to handle, the lightweight nature of this project makes it accessible anywhere, instantly giving you the overview of the 2026 season.

## 🌟 Key Features

*   **Live Race Countdown**: A precision countdown tracking the exact days, hours, minutes, and seconds until the lights go out for the next Grand Prix.
*   **Driver & Constructor Standings**: The current driver and constructor championship battle visualized in real-time-ish fashion.
*   **The 23-Round Calendar**: A horizontal progress strip tracking past, current, and upcoming races alongside race winners and flag emojis.
*   **Paddock Intel**: Handpicked dynamic news and paddock information presented in an editorial magazine design format.
*   **Personalization Highlights**: Quickly see stats like the fastest laps, fastest pit stops, and personal favorites right at the top.

## 🛠️ Technology Stack

*   **HTML5** for raw accessible structure
*   **Vanilla CSS3** for modern, dynamic aesthetics and layout styling
*   **Vanilla JavaScript** for handling the live countdown timers, visual micro-animations, and routing logic

## 🚀 How to Use

Because it doesn't run on any framework, launching your Pit Wall is simple!

1. Clone or download this repository.
2. Open `index.html` in any modern web browser (Edge, Chrome, Chrome, etc.).
3. Enjoy the aesthetic race data.

*Lights out and away we go!*
